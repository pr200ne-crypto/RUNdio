### 1日の流れ（関連部分）

朝または夕方に**一人でラン**。イヤホンをし、**スマホ**でアプリを操作しつつ走る。別デバイスで記録することもあるが**アプリ単体でも完走**したい。

### 課題・ペイン

- 練習が**マンネリ**し、走る前から気分が上がらないことがある。  
- 走行中は**数値だけでは物足りず**、**励ましや番組のような流れ**が欲しい。  
- 走ったあと**次を何にするか**を考えるのが面倒。

### 現在の解決方法

記録アプリ、腕時計、ラジオ／ポッドキャストを併用。ルートは慣れたコースが多い。 [仮]

### 求めているもの

- **今日の一本**を決めやすく（ルート上の**条件検索**）。  
- 走っている間**自然な音声**で伴走。  
- **ご褒美やゴール**に紐づく一言でテンション向上。  
- **距離**と**時間**、好きな方で**区切り**を取りたい。

### このプロダクトを使う瞬間

走る前の計画時、イヤホンを装着してスタートした直後〜ゴール手前、走行直後。

### 情報収集の方法

アプリストア検索、SNS、ランナー仲間の口コミ。 [仮]
