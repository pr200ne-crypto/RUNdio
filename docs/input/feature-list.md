## 機能詳細（例：条件検索）

**概要：** ルート上の前方距離を基準に、カテゴリ条件で候補を列挙し、ユーザーがチェックポイントを選ぶ。

**ユーザーストーリー：**  
「ランナーとして、今日のルートの先に**銭湯**を組み込みたい。なぜなら**ゴール後のご褒美**で走る動機が上がるから。」

**入力：** ルート、検索カテゴリ、前方距離の目安 等  
**出力：** 候補リスト、地図上の位置 等  
**受け入れ条件：** [詳細要件で記述] [仮]
