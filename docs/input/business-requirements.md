# ビジネス要件（ランディオ）

## 1. 課題定義

### 解決する課題

ソロランナーが、**練習内容のマンネリ**と**モチベーションの低下**を抱え、走行中に**楽しさや伴走感**を得にくいこと。

### 課題の深刻度

継続走者にとっては日常的な摩擦。解消すると**継続率・満足度**に直結しうる。 [仮]

### 課題が発生する状況

一人で走るセッションの前（何をどこまで走るか決めづらい）、走行中（単調さ・孤独感）、走行直後（次に何をすべきか不明瞭）。

---

## 2. 市場環境

### 既存の代替手段

一般的なランニング記録アプリ、腕時計の音声アラート、ポッドキャスト／ラジオ、自己計画のルートアプリなど。

### 既存手段の課題

記録・数値に偏り**感情・動機**への支援が弱い、または**ルート上の「今日の一本」と音声体験が一体**になっていない場合がある。 [仮]

### 競合サービス

（要調査）Strava、Nike Run Club、ASICS Runkeeper、国内ルートアプリ等。差別化軸は別途整理。 [仮]

---

## 3. 差別化ポイント

### 競争優位性（方向性）

- **プランニング**（ルート上の条件検索）と**走行中のパーソナルラジオ的音声**、**走後提案**の三位一体。  
- **距離／時間**の好みに合わせた**区切り設計**。  
- **自然な人間的な音声体験**を最優先の品質目標に置く。
